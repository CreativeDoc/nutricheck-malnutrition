# NutriCheck - Mangelernährungs-Screening

Professionelles Screening-Tool zur Erkennung von Mangelernährung nach NRS 2002.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Development

```sh
npm install
npm run dev
```

## Deployment

Hosted on Vercel. Push to `main` triggers auto-deploy.
